Get Started in Minutes

Deploy Right API and start connecting backends to Claude.ai

1

Deploy with Docker

git clone https://github.com/right-api/gateway
cd gateway
docker-compose up -d
2

Configure Claude.ai

# Add MCP Server in Claude.ai:
URL: https://your-domain.com/mcp
Type: HTTP
Headers: X-Gateway-Token: your-token
3

Start Automating!
